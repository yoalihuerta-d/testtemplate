## A. Glosario de Términos
 
:::{note}
Esta sección define conceptos clave mencionados en todo el libro. Consulta este glosario cuando encuentres un término técnico desconocido.
:::
 
### Términos Fundamentales
 
**Algoritmo**
: Un procedimiento paso a paso para resolver un problema o completar una tarea. En programación, es un conjunto de instrucciones lógicas que sigue una computadora.
 
**Variable**
: Un contenedor nombrado que almacena un valor. Las variables permiten guardar datos que pueden cambiar durante la ejecución del programa.
 
**Función**
: Un bloque reutilizable de código que realiza una tarea específica. Las funciones ayudan a organizar el código y reducir la repetición.
 
**Iteración**
: El proceso de repetir un conjunto de instrucciones múltiples veces, generalmente usando bucles `for` o `while`.
 
**Librería/Módulo**
: Un conjunto de código preescrito que proporciona funcionalidades específicas. Los módulos pueden importarse en tus programas para reutilizar código.
